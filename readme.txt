A customizable floating widget for Abjad (Ebced), Bastet, and Huddam calculations with support for Arabic, Hebrew, and Turkish.

== Description ==

**Abjad Widget** adds a beautiful floating calculator to your WordPress site, enabling mystical and numerological calculations:

* 🔮 **Abjad Calculation** - Traditional Arabic alphanumeric system
* 📿 **Bastet Calculation** - Advanced numeric transformations
* 👤 **Huddam Generation** - Spiritual entity name generation
* 🌍 **Multi-language Support** - Arabic, Hebrew, Turkish, English
* 🎨 **Customizable** - Colors, positions, button styles (emoji, SVG, PNG)
* ⌨️ **Virtual Keyboard** - Built-in keyboard for easy input

= Features =

* Real-time calculations with live preview
* Multiple calculation methods (Hisab al-Jumal, Gematria, Turkish Abjad, etc.)
* Element classification (Fire, Air, Water, Earth)
* Shadda support for Arabic letters
* Responsive design
* Buy Me a Coffee integration

= How to Use =

1. Install and activate the plugin
2. Go to Settings → Abjad Widget
3. Configure your button appearance and Buy Me a Coffee ID
4. The widget will appear on your site as a floating button

= Support =

For support, feature requests, or bug reports, please visit the [plugin support forum](https://wordpress.org/support/plugin/abjad-widget).

== Installation ==

1. Upload the `abjad-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings → Abjad Widget to configure
4. Add your Buy Me a Coffee ID and customize colors
5. The widget will automatically appear on your site

== Frequently Asked Questions ==

= What is Abjad? =

Abjad is an alphanumeric code where Arabic letters have numerical values, used in numerology and Islamic mysticism.

= Can I use Hebrew or Turkish letters? =

Yes! The widget automatically detects the script type (Arabic, Hebrew, or Turkish) and applies appropriate calculations.

= How do I add my Buy Me a Coffee button? =

Enter your Buy Me a Coffee username in the settings. The widget will automatically create a support tab with your custom button.

= Can I customize the button appearance? =

Absolutely! You can use emoji, custom SVG, or PNG images for the button. Also customize color and position.
